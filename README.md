# Smack Magick

A single-page site for Smack Magick — iridescent hand-knit jellyfish, made one stitch at a time by Anna.

## Run locally

It's a single static HTML file. Open `index.html` in a browser, or serve the folder with anything:

```bash
python3 -m http.server 8000
# then open http://localhost:8000
```

## Deploy

Enable GitHub Pages in repo **Settings → Pages** — source: `main` branch, root directory. The site will be live at `https://<username>.github.io/SmackMagick/`.

## Assets

- `index.html` — the site
- `jelly.png` — jellyfish mark (used in the nav)
- `new logo trans.png` — combined logo with wordmark (used in the footer)
- other PNGs are extra logo variants kept in the repo for reference
